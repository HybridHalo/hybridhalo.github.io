Overview

What it is

Recommender is a personal curation site — a handpicked list of favourite things presented in a clean, readable format. Rather than a social feed or algorithmic ranking, it's a deliberate, editor-curated collection: the things worth actually recommending to someone you trust.

The design uses Libre Baskerville serif for body text and Work Sans for UI elements, with a warm gradient dark background and a fine grid overlay. The typographic choices give it a literary, magazine-like quality that feels intentional rather than template-driven.

Built with React and deployed as a standalone page, it requires no backend — content lives directly in the component and the whole thing runs without a server.
