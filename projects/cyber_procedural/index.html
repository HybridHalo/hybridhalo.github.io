Overview

What it is

This is a self-contained generative art gallery — no libraries, no dependencies, just a single HTML file with 500 hand-written canvas algorithms. Every piece in the collection follows the same strict aesthetic: pure black background, near-invisible detail, monospace typography. The "Black Series" format forces each algorithm to express itself through structure alone rather than colour.

The gallery uses scroll-snap so each piece occupies the full viewport, one at a time. A seed input at the top lets you rerun any algorithm with a custom string, producing a different but deterministic output — the same seed always produces the same image. This makes it easy to save and share specific variations you like.

Subjects range from mathematical structures (contour lines, Voronoi cells, flow fields) to hardware-inspired forms (PCB traces, chip die layouts, circuit grids) to pure signal processing visualisations. Each algorithm is a standalone study.
